# Docker Interactive Learning

This repository contains interactive challenges to help you learn Docker concepts.

## Challenges

### Anonymous Exit
- **Description:** Debug a container that exits immediately with code 127, exploring common `ENTRYPOINT`/`CMD` and script issues.
- **Difficulty:** Easy
- **Tags:** Docker, CLI, Debugging
- **Path:** challenges/docker/dc-anonymous-exit.html

### Isolated Service
- **Description:** Diagnose a networking problem where containers on the same Docker network cannot communicate, focusing on network configuration and DNS resolution.
- **Difficulty:** Medium
- **Tags:** Docker, Networking, CLI
- **Path:** challenges/docker/dc-isolated-service.html
